# ksenija.co

Personal website.

## Deploy

1. Go to **Settings** → **Pages**
2. Under **Source**, select **main** branch
3. Save
4. Your site will be live at `https://yourusername.github.io/ksenija.co`

To use custom domain (ksenija.co):
1. In **Settings** → **Pages** → **Custom domain**, enter `ksenija.co`
2. Add these DNS records at your domain provider:
   - Type: **A** → `185.199.108.153`
   - Type: **A** → `185.199.109.153`
   - Type: **A** → `185.199.110.153`
   - Type: **A** → `185.199.111.153`
   - Type: **CNAME** → `www` → `yourusername.github.io`
